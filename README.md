# sweetDiscount

Functions to apply sales discounts (2 for 1, spend x save y, pack discount)

## Requirements

* Node 19.0.1
* Git

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/Menalujan/sweetDiscount.git
cd test
```

```bash
npm install
```

## Steps for access

To start run the following

```bash
npm start
```
